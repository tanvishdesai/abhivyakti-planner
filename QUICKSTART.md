## Getting Help

### If events won't load:
1. Check internet connection
2. Refresh browser (F5)
3. Clear browser cache
4. Close and reopen application

### If filters aren't working:
1. Click "Clear" to reset
2. Try filtering one thing at a time
3. Refresh the page

### If combos aren't showing:
1. Make sure you have multiple events on same date
2. Check if events have enough time gap
3. Verify venue locations
